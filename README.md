

# 🌐 Personal Portfolio Website

## 👩‍💻 About the Project

This is a responsive and interactive personal portfolio website built using **HTML, CSS, and JavaScript**. It showcases my skills, projects, education, and contact information in a clean and modern UI.

The goal of this project is to present my work and technical abilities in a structured and visually appealing way.

---

## 🚀 Features

* Responsive design for desktop and mobile
* Smooth navigation with fixed navbar
* Hero section with typing animation
* About section with professional summary
* Skills section with interactive cards
* Projects section with hover effects and popup links
* Education and certifications section
* Contact section with social links
* Clean UI with gradient background and modern styling

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Flexbox, Grid, Animations
* **Icons:** Font Awesome
* **Fonts:** Google Fonts (Poppins)
